Ejemplos
Variables y salida
zenit x := 10.
zenting msg := "Hola Mundo".
show(msg).

Lectura de datos (read)
zenit edad := 0.
read(edad).
show("Tu edad es:").
show(edad).
Condicional múltiple
zenit nota := 75.
when (nota >=? 90) {
  show("Excelente").
} otherwise when (nota >=? 70) {
  show("Aprobado").
} otherwise {
  show("Reprobado").
}
Bucle contador
zenit i := 1.
loop (i <=? 5) {
  show(i).
  i := i + 1.
}
Suma acumulativa
zenit suma := 0.
zenit n := 1.
loop (n <=? 10) {
  suma := suma + n.
  n := n + 1.
}
show("Suma 1-10:").
show(suma).
Precedencia (2 + 3 * 4 = 14)
zenit a := 2 + 3 * 4.
show(a).
Gramática Formal (BNF)
<programa> ::= <lista_instrucciones>
<instruccion> ::= <declaracion> | <asignacion> | <condicional> | <ciclo> | <salida> | <entrada>
<declaracion> ::= <tipo_dato> <id> ":=" <expresion> "."
<asignacion> ::= <id> ":=" <expresion> "."
<salida> ::= "show" "(" <expresion> ")" "."
<entrada> ::= "read" "(" <id> ")" "."
<condicional> ::= "when" "(" <condicion> ")" "{" <lista> "}" <alternativa>
<alternativa> ::= "otherwise" <condicional> | "otherwise" "{" <lista> "}" | ε
<ciclo> ::= "loop" "(" <condicion> ")" "{" <lista> "}"
<tipo_dato> ::= "zenit" | "zenloat" | "zenting" | "zenbool"
